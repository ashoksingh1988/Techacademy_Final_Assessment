5. TROUBLESHOOTING TIPS
================================================================================

‚ùå PROBLEM: "Java is not recognized"
‚úÖ SOLUTION:
   - Install Java JDK 11 or higher
   - Add Java to PATH environment variable
   - Restart terminal

‚ùå PROBLEM: "Maven is not recognized"
‚úÖ SOLUTION:
   - Install Apache Maven
   - Add Maven to PATH environment variable
   - Restart terminal

‚ùå PROBLEM: "Python is not recognized"
‚úÖ SOLUTION:
   - Install Python 3.13+ (or 3.8+)
   - During installation, check "Add Python to PATH"
   - Restart terminal

‚ùå PROBLEM: "Appium server is not running"
‚úÖ SOLUTION:
   - Install Appium: npm install -g appium
   - Start Appium server: appium
   - Verify: http://127.0.0.1:4723/status

‚ùå PROBLEM: "No Android device detected"
‚úÖ SOLUTION:
   - Connect Android device via USB
   - Enable USB Debugging on device
   - Run: adb devices
   - OR start Android emulator

‚ùå PROBLEM: "Module not found" (Python)
‚úÖ SOLUTION:
   - Navigate to python-selenium-automation
   - Run: py -m pip install -r requirements.txt

‚ùå PROBLEM: "Browser driver not found"
‚úÖ SOLUTION:
   - Java frameworks use WebDriverManager (automatic)
   - Python framework uses webdriver-manager (automatic)
   - Internet connection required for first run

‚ùå PROBLEM: "Tests fail immediately"
‚úÖ SOLUTION:
   - Check internet connection
   - Verify application URL is accessible
   - Check browser installation
   - Review logs in target/surefire-reports/

‚ùå PROBLEM: "Permission denied on bat file"
‚úÖ SOLUTION:
   - Run PowerShell as Administrator
   - OR navigate to folder and double-click bat file
   - OR use Maven/Python commands directly

‚ùå PROBLEM: "Report not generated"
‚úÖ SOLUTION:
   - Check reports/ folder in framework directory
   - Ensure tests completed execution
   - Check file permissions on reports/ folder
